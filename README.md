Provides a Menu Vue component to be used within a Druxt (DRUpal nuXT) project.

## Install

`$ npm install druxt-menu`

## Usage

Add module to `nuxt.config.js`

```js
module.exports = {
    ...
    'druxt-menu'
  ]
}
```
